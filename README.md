# chrome-ext

Simple template of chrome-extention


* Run FastAPI-server:
```bash
uvicorn main:app --reload --port 3400
```

Requirements:
```bash
python-dotenv
mysql-connector-python
requests
fastapi
uvicorn
python-jose[cryptography]
authlib
pyjwt
itsdangerous
google-auth
```

`pip install "python-jose[cryptography]"`
